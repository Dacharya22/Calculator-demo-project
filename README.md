🧮 Modern Calculator

A simple and responsive calculator built using **HTML, CSS, and JavaScript** during my early web development learning journey.

This project helped me understand how to build interactive user interfaces, manipulate the DOM, and implement basic calculator logic using JavaScript.

 ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📊 Percentage calculation
* 🗑️ Clear all (AC)
* ⌫ Delete last input (DEL)
* 🎨 Modern gradient UI design

🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

 🎯 What I Learned

* DOM Manipulation
* Event Listeners
* JavaScript Logic Building
* UI Design Basics
* Interactive Web Development

 🚀 Future Improvements

* Add keyboard support
* Improve mobile responsiveness
* Add error handling
* Replace `eval()` with a safer calculation method
* Add button animations
